
Dans /GetStarted/ :
    make reset : Redémarre Docker-Compose
    make tty : Commande bash dans la console
    make start : Démarre Docker-Compose
    make stop : Stop Docker-Compose

    Afficher page PHP dans le navigateur :
    http://localhost:8080/public/NomDeLaPage

    http://localhost:8081/index.php?route=/server/databases

    http://localhost:8080/public/info.php
